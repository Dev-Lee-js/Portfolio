Yeonda featured work #1

  • #개요

    • '(마음을) 연 다(이어리)' 라는 캐치 프레이즈에서 따온 '연다 Yeonda' 는 기본적으로 일기 작성 및 관리에 대한 서비스와 일기를 통해 다른 이들과 인연을 맺을 수 있는 기회를 제공합니다. 보통 소개팅 서비스라 함은 상대방의 외적인 모습을 중점으로 매칭이 됩니다. 근데 상대방의 내면을 보고 매칭을 시켜주면 더 좋지 않을까? 하는 생각으로 개발된 서비스입니다.

  • #특징

    • 로그인시 Access-Token과 Refreash-Token으로 인증정보를 쿠키에 저장합니다.

    사용자가 선호하는 성별, 거리, 나이대를 가진 작성자가 작성한 일기를 랜덤으로 보여줍니다.

    Open AI를 활용해서 사용자가 작성한 글을 읽고 관련성이 높은 태그들을 태깅해줍니다.

    무한 스크롤을 구현했습니다.

  • #활용 라이브러리 & API

    • Typescript, Socket.io, Express, React, Axios, JWT, NodeMailer,
    Tailwind CSS, Tanstack Query, React Hook Form, Zustand

Current Page:#01

튜토리얼

처음 접속하면 전체 서비스들에 대한 소개가 있는 튜토리얼 페이지로 이동합니다.
튜토리얼 페이지를 모두 확인하면 로그인 페이지로 이동합니다.

로그인

Email 인증을 통해 회원가입/비밀번호 찾기를 수행할 수 있고, 계정정보 뿐만 아니라, 성별, 연령대, 거리 등 매칭 상대의 취향을 선택할 수 있습니다.
로그인시 Access-Token과 Refreash-Token으로 인증정보를 쿠키에 저장합니다.

회원가입

비밀번호 찾기

추천 일기 페이지

사용자가 선호하는 성별, 거리, 나이대를 가진 작성자가 작성한 일기를 랜덤으로 보여줍니다.

인기 일기 페이지

기본적으로 최근 좋아요를 많이 받은 5개의 일기 목록을 보여줍니다.

일기 작성 페이지

사용자는 제목과 내용 작성에만 집중할 수 있고 태깅은 Open AI 엠베딩과 데이터 베이스 내 태그 목록과 연계해 자동으로 이루어집니다

내 일기 페이지

tanstack/react-query에 useInfiniteQuery를 사용해서 무한 스크롤을 구현했습니다. 최신 날짜순, 좋아요 많은 순 정렬 기능이 있습니다.

일기 상세보기 페이지

조회, 수정, 삭제 등 기본적인 동작이 가능합니다. 첫 작성시 자동 태깅되었던 태그들을 자신이 원하는 태그로도 수정할 수 있습니다.

채팅 페이지

채팅 페이지에서는 서로 연결된 사용자간 채팅이 가능합니다.

관리자 페이지

관리자 페이지에서는 이용자 분석, 통계를 확인하거나 태그 설정을 할 수 있습니다.

에러 페이지

잘못된 주소로 접속을 요청하거나, 에러가 발생한 경우 에러 페이지로 이동시켜줍니다.
